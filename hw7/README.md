## 網站操作指南
1. 在 /backend 資料夾中複製 .env.defaults 並改名成 .env
2. 在 .env 檔案中貼上 mongodb 的網址
3. cd 到 /frontend 並執行 yarn install 或是 npm install
4. cd 到 /backend 並執行 yarn install 或是 npm install
5. 在 hw7 資料夾執行 yarn start 和 yarn server 即可
